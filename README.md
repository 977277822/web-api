## kissy-combo
    kissy请求合并，目前只支持kissy-base文件的请求合并，后面会陆续补充kissy自定义模块的文件合并
** 文件目录结构 **
* bulid （文件夹）
    合并文件缓存目录
* src （文件夹）
    kissy-base
    kissy-mods 自定义模块目录
* index.js （combo服务启动文件）
* pathmap.json （关联映射）
    缓存目录与请求映射文件
